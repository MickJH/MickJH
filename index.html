Projecten Op school en werk heb ik aan diverse projecten gewerkt. Hieronder een aantal voorbeelden, meer projecten zijn te vinden op mijn Github.

Project Server Side Web Frameworks

Dit was een individueel project waarbij we gebruik moesten maken van het Microsoft ASP.NET framework en C# in combinatie met een SQL Database en het toepassen van de Onion Architecture. De casus ging over dat mensen die moeite hadden met de Nederlandse taal samen konden komen op spelavonden om zo op een gezellige manier de taal te leren. De website werd destijds gehost op azure, meer informatie kun je vinden op de github pagina.

Github

Project Client Side Web Frameworks

Dit was wederom een individueel project waarbij we gebruik moesten maken van het Angular framework en Typescript in combinatie met een NoSQL Database (MongoDB) en een Grahph database (neo4j). De casus mocht jezelf verzinnen zolang je maar genoeg entiteiten had en gebruikers had die elkaar kunnen volgen zodat je op basis daarvan recommendaties kan maken voor de gebruiker. Meer informatie kun je vinden op de github pagina.

Github

Project The Circle

Dit project was een groepsproject die geinspireerd was op de film en het boek: The Circle van Dave Eggers. Het project is vooral gericht op het altijd kunnen livestreamen en het volgen van de livestreamers. Ik heb hierbij vooral gefocust samen met een klasgenoot op het streamen door middel van WebRTC en SignalR en het veilig verzenden daarvan om integriteit te kunnen waarborgen door middel van encryptie en keys. Meer informatie kun je vinden op de github pagina.

Github

Project Software ontwerp & Architectuur

Dit project was een duo project waarbij we meerdere Creational patterns en Structural patterns moesten toepassen in een project. Gezien de korte tijdsduur van het project is het uitgewerkt met simpele log voorbeelden. Wij hebben de volgende patterns weten toe te passen: Factory pattern, Adapter pattern, Composite pattern, Strategy pattern, Observer pattern, State pattern & de Template pattern. Vond dit zelf een heel interessant project om aan te werken, meer informatie is uiteraard weer te vinden op github pagina.

Github

Project Naamgeving

Bij Zorgwaard speelde het probleem dat we voor verschillende applicaties meerdere inlognamen hadden. Dit was uiteraard niet handig en wilde dan ook voor alle applicaties een uniforme naamgeving. Dit project was voor het omzetten van de inlognamen van alle medewerkers, eerst deden we dit met de medewerker erbij wat er voor zorgde dat we niet veel namen konden omzetten. Uiteindelijk ben ik met het idee gekomen om de naamgevingen online om te zetten en heb dit dan ook mogen pitchen aan de directie. Dit idee is uiteindelijk doorgevoerd waardoor we dit project een stuk sneller en efficienter hebben kunnen afronden.
